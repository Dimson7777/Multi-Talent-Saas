# Nexus — Multi-Tenant SaaS Platform

A modern multi-tenant SaaS platform focused on authentication, billing workflows, team management, and real-time operational visibility.

Built with a premium product-first UI inspired by modern SaaS platforms like Linear, Vercel, and Stripe.

---

## ✨ Features

- Multi-tenant workspace architecture
- Secure authentication & session management
- Role-based access control (RBAC)
- Team invitation system
- Stripe billing integration (test mode)
- Real-time activity timeline
- Modern premium dashboard UI
- Responsive dark-mode interface
- Floating command-center inspired design
- Smooth animations and micro interactions

---

## 🛠 Tech Stack

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Dimson7777/Multi-Talent-Saas.git
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start development server

```bash
npm run dev
```

---

## 🔐 Environment Variables

Create a `.env.local` file:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
VITE_STRIPE_PUBLIC_KEY=your_stripe_public_key
```

---

## 📦 Build

```bash
npm run build
```
